# ecommerce-sales-analysis-python.
# E-commerce Sales Analysis with Python

This project explores e-commerce transaction data using Python and Jupyter Notebook.

## Project goal
The goal of this project is to analyze sales data and find useful business insights such as:
- Monthly revenue trends
- Top-selling products
- Top countries by revenue
- Top customers by revenue

## Dataset
The dataset used in this project is the Online Retail dataset.

## Tools used
- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- NumPy

## Files
- `sales-analysis.ipynb` — the main analysis notebook
- `README.md` — project overview
- `data/OnlineRetail.csv` — dataset file

## How to run
1. Open the project folder in Jupyter Notebook.
